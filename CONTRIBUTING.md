# Contributing

Thanks for helping improve AutoCIP.

## Before You Start

1. Open an issue to discuss non-trivial changes.
2. Keep pull requests focused and small where possible.
3. Include tests for behavior changes.

## Development

```bash
uv sync --all-extras
uv run ruff check .
uv run pytest -q
```
